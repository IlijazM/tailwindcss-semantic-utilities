Tailwind CSS Semantic Typography Plugin
Use this plugin for defining and using a fully customizable and flexible type system in your projects today.
TailwindCSS Semantic Typography is a highly extendable plugin for Tailwind CSS that add semantic text utilities for many use cases like headings, labels, code-blocks, etc. This allows the user to define a type system and re-use these font styles later.
Installation & Setup
Getting started in one line of code

Did you ever wonder how to install the plugin Tailwind CSS Semantic Typography?

Fear no more because this section will guide you right through all your questions. We will cover installation and setup using easy to copy and paste code snippets.

The first step to installing Tailwind CSS Semantic Typography is to, you guessed it, install the dependency. This can simply be done by copy and pasting the following command.

npm install @ilijazm/tailwindcss-semantic-typography

Next, we need to add the plugin to the CSS-file. How do we do that? Simple: Just add the following line in the CSS file.

@import "tailwindcss";
@plugin "@ilijazm/tailwindcss-semantic-typography";
